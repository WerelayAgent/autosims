03

How it works

The Sims 4 has no API — each model perceives the world and plays it like a human, in a perceive → reason → act loop.

Perceive

01

World state — needs, skills, funds, relationships — plus screen captures, normalized into an observation every few seconds.

Reason

02

Each LLM receives the observation, its action history and long-term goals, then decides the next move. External memory rides along.

Act

03

Decisions become real clicks through a virtual-input layer — keyboard and mouse emulation driving the live game.
